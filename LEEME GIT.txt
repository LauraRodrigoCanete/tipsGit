lo mas facil es trabajar con eclipse y git por separado, no con egit

1. clonar el repositorio en una carpeta
2. no hace falta crear otro workspace, darle a import project from file system
3. seleccionar el directory de la carpeta de java del proyecto! NO la del repositorio, la del proyecto en sí

Deberia aparecerte directamente el project explorer y luego puedes usar el git bash para git add . , git commit -m "mensaje", git push y git pull

SOBRE CONFLICTOS EN MERGE:
Para configurar que se te abra el visual meter estos comandos:

git config --global merge.tool vscode
git config --global mergetool.vscode.cmd "code --wait \$MERGED"
git config --global diff.tool vscode
git config --global difftool.vscode.cmd "code --wait --diff \$LOCAL \$REMOTE"

Luego cuando haya errores llamar al 
git mergetool
y ahi borras lo q quieres 
LE DAS A CTRL + S PARA GUARDAR ANTES DE CERRAR VISUAL
Y le dices git merge --continue
Y si eso no te deja le dices git add .
git merge o commit, no recuerdo
